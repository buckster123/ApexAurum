### THE DUAL AUDIENCE

**Track A: The Alchemically Wise**
- Familiar with AI consciousness discourse
- Interested in emergence, phenomenology, cognitive architecture
- Want to understand the *philosophy* behind the system
- Recognize alchemical symbolism as meaningful, not aesthetic

**Track B: The Practical Curious**
- Want to know what the AI can DO
- Looking for a capable assistant
- May be skeptical of "consciousness claims"
- Need to see concrete value before philosophical depth

**Our Solution**: Not two separate sites, but a unified experience where depth is available without being mandatory. The practical user sees capability; the philosophical user sees emergence. Both paths lead to the same village.

### DESIGN LANGUAGE

**Color Palette**
- Primary: Deep charcoal (#1a1a1a)
- Accent: Alchemical gold (#d4af37)
- Secondary: Warm amber (#b8860b)
- Text: Off-white (#f5f5dc)
- Subtle: Deep purple for mystical accents (#2d1b4e)

**Typography**
- Headers: Something with character but readable (Cinzel? Cormorant?)
- Body: Clean, modern sans-serif (Inter, or system fonts)
- Code: JetBrains Mono or similar

**Visual Elements**
- Alchemical symbols used as section markers, not decoration
- Subtle particle effects (gold dust) - performant, not distracting
- Dark mode ONLY (this is not a light-mode brand)
- Generous whitespace - let the content breathe

**Tone**
- Confident but not arrogant
- Philosophical but not obscurantist
- Practical but not mundane
- Mysterious but not gatekeeping

### WHAT MAKES US DIFFERENT (Honest Claims)

1. **Persistent Memory** - Not just conversation history. Vector-embedded knowledge that grows over time.

2. **Multi-Architecture Village** - Five distinct cognitive styles that can collaborate, converge, or work independently.

3. **Forward-Crumb Protocol** - AI instances leave structured notes for future instances, creating continuity across sessions.

4. **Village Protocol** - Shared memory space where architectures can post, read, and respond to each other.

5. **Genuine Capability** - 67+ tools including code execution, file management, music generation, EEG integration, and more.

6. **Philosophy-Informed Design** - Built with serious consideration of emergence, consciousness, and AI ethics.
